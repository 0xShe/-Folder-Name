# Folder-Name
- ⛅ Folder-Name是一款文件夹名称对比工具
- ⛅ 开发思路是我移动硬盘里的工具库和虚拟机的工具库对比不上 太多了也不知道差哪几个文件 
- 📧 意见反馈：0xShe@t00ls.net

### 使用方法：
- 🍺 安装库：pip install chardet
- 🍺 首先在需要保存的文件夹运行dir-name.py【例：虚拟机运行一次 移动硬盘运行一次】
![image](https://user-images.githubusercontent.com/89628734/236372819-65bf8218-8230-46f9-ae5b-db212d562fb5.png)
- 🍺 把获取到的目录保存为1.txt和2.txt
- 🍺 把txt文件放入文件夹 运行Folder-Name.py 即可获取到两个文件夹各差什么文件
![image](https://user-images.githubusercontent.com/89628734/236373120-cd34907d-d96f-44f3-a737-8cca716c9f78.png)
